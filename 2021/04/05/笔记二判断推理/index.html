入门课主要将前三点

1. 翻译推理

三个角度类比推理规则（选则一个理解即可）：

  1. 集合的角度：

    1. A -> B ，表示 集合A 比 B小、A属于B、A在B中、瓜子A在瓜B中、Ａ的老大是Ｂ
    2. !B -> !A ，表示 没有B就没有A、A在B中、没有瓜B就没有瓜子A
    3. 只有B才A ，表示 B比A大、没有B就没有A
    4. B是A的关键， 表示 有B才有A，B比A大、

  2. 树、决策树的角度（树根的形状、越往上越抽象，范围越大）

    1. A -> B ，表示 A是B的子节点、B是A的父节点、　->指向父节点
    2. !B->!A ， 表示 不经过B就到不了A、B是A的父节点
    3. 只有B才A ，表示Ｂ是Ａ的父节点
    4. B是A的关键，表示Ｂ是A的父节点、不经过Ｂ就到不了Ａ
    • 演绎推理：从上推到下，从前提推到结果，所以是必然推理
    • 归纳推理：从下推到上
    • 类比推理：从一个分支跳跃另一个分支

  3. 逻辑学角度（外延＝元素、内涵＝条件，限制）

    1. A -> B ，表示　Ａ的外延比Ｂ少，（Ａ的内涵比Ｂ多）
    2. !B -> !A ，表示　没有Ｂ的外延就没有Ａ的外延
    3. 只有B才A ，表示　Ｂ的外延比Ａ多，　没有Ｂ的外延就没有Ａ的外延
    4. B是A的关键，表示　。。。。懒得写了

  • 词项的内涵是指该词项所反映的对象的本质属性的总和，也即对事物的特有属性的反映；
  • 词项的外延是指具有该词项所反映的本质属性的一切对象
  • 内涵 = 限定词。限定词越少，约束范围越小，包含内容/实体越大。
  • 外延 = 实体。外延越小、实体越少，因为限定词约束得多，内涵越大。
